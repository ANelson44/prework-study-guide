HTML

  • HTML - Hypertext Markup Language. Is used to create the structure of a webpage.
  • Element is a type of content on a webpage. Four commonly used elements are Text elements, Image elements, Header elements, and List elements (like the list we have created here)
  • An HTML attribute defines additional information about an element.
  • HTML attributes are always defined in the beginning element, never the closing element.
  • The head element contains information about the webpage.
  • The body element represents the visibile content shown to the user.
